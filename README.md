  ## Description:
  A REACT application which  allows users to match apply and post jobs based on required skills.

  ## Technologies:
  - REACT
  - NODEJS
  - Expressjs
  - HTML
  - CSS
  - GRAPHQL
  - GRAPHQL-UPLOADS
  - APOLLO-Server
